# 外卖管理系统简介 python474

## 项目概述

外卖管理系统是基于Python语言和SQL Server数据库实现的一套在线订餐管理平台。该系统涵盖了用户、商家、外卖员以及店铺等多个方面的管理功能，实现了从顾客下单到外卖员配送的完整业务流程。

## 技术栈

- 后端：Python
- 数据库：SQL Server

## 功能模块

### 管理类功能

- 用户管理
- 商家管理
- 外卖员管理
- 店铺管理（包含菜品管理）

### 流程类功能

- 更新各类状态（如订单、外卖员、商家）
- 派单模式与抢单模式切换

### 浏览查询类

- 用户模糊查询菜品和店铺
- 商家查询外卖员
- 用户查询订单配送状态
